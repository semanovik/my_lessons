Отличительные особенности кода, экспортированного из рекордера, по сравнению с кодом, написанным вручную:

1.Рекодер использует для поиска элементов способ с объектом By
2.Не создает/хранит объекты (скорее всего по той причине, что не сможет присвоить им логичные названия)
3.Используется «setup/teardown» для функций и методов
4.В assert в случае несоответсвия не указывает сообщение из поля "Description". Иными словами, при запуске кода не увидим человекопонятное описание ошибки
5.Код теста не удовлетворяет требованиям к code style
